# AI Job Search Agent 🚀

Tired of spending 30–60 minutes tailoring your resume and writing a cover letter for every job application?

AI Job Search Agent automates the most repetitive parts of the job search process.

## What it does

Simply open a LinkedIn job posting and click a button.

The agent automatically:

✅ Extracts job details from LinkedIn
✅ Analyzes job requirements
✅ Calculates a Resume Match Score
✅ Identifies keyword gaps and missing skills
✅ Generates a tailored ATS-friendly resume
✅ Creates a customized cover letter
✅ Exports everything as DOCX and PDF
✅ Produces a detailed match analysis report

All in under a minute.

## Tech Stack

* Chrome Extension (Manifest V3)
* FastAPI
* OpenAI GPT
* Python
* DOCX/PDF Generation
* LinkedIn Job Parsing

## Example Output

For every application, the agent generates:

📄 Tailored Resume
📄 Customized Cover Letter
📄 Match Analysis Report
📄 PDF Versions of all documents
